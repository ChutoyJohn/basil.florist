What this is

So I got bored and decided to set up a 24/7 TF2 server.
I've been wanting to see how popular a purely vanilla, no sourcemod, no custom maps, nothing TF2 server would fair in the world of ultra moderated ultra modded community servers

This experiment was a success for the first few days but gradually became a failure due to lack of maintenance and interest.
I will keep the current server going for as long as I'm interested in keeping it up.
